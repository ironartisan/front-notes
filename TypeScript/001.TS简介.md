# TypeScript 简介

## 什么是 TypeScript

- 是以JavaScript 为基础构建的语言
- 可以在任何支持 JavaScript 的平台上执行
- TS 不能直接被 JS 解析器直接执行
- 一个 JavaScript 的超集
- TS 扩展了 JavaScript, 并添加了类型

![20230106173331](https://cdn.jsdelivr.net/gh/ironartisan/picRepo/20230106173331.png)


## TS的优势
- 增加了一些 JS 中没有的类型
- 支持ES的新特性
- 添加了ES不具有的新特性, 抽象类、接口、工具、装饰器等
- 具有丰富的配置选项
- 强大的开发工具
